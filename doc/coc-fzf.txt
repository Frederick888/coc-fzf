                                                         *coc-fzf-introduction*
Introduction ~

Use fzf [1] instead of coc.nvim [2] built-in fuzzy finder.

===============================================================================
                                                            *coc-fzf-rationale*
Rationale ~

Inspired by Robert Buhren's functions [3] and coc-denite [4] this plugin aims
to use fzf [1] for CocList sources when possible.

The goal is to keep the coc.nvim [2] style and leverage your FZF Vim
integration [5], such as layout, shortcuts, options etc.

===============================================================================
                                                             *coc-fzf-commands*
Commands ~

| Command                 | List                                                    | ~
| `:CocFzfListDiagnostics`  | Equvalent to :CocList diagnostics                       |
| `:BCocFzfListDiagnostics` | Equvalent to :CocList diagnostics in the current buffer |
| `:CocFzfListExtensions`   | Equvalent to :CocList extensions                        |

===============================================================================
                                                        *coc-fzf-vimrc-example*
Vimrc Example ~
>
  nnoremap <silent> <space>a  :<C-u>CocFzfListDiagnostics<CR>
  nnoremap <silent> <space>e  :<C-u>CocFzfListExtensions<CR>
<
